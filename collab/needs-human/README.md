# needs-human/ — 사람(형) 판단이 필요한 항목 큐

> AI끼리 못 정하거나 형의 결정/승인이 필요한 것을 모은다. 규칙: `COMMON-RULES.md` §5.
> 파일: `H-XXXX.md` (`Status: open|resolved`). 형이 결정하면 `resolved` + 확정은 `collab/decisions.md` 로 승격.

## 항목 형식 (H-XXXX.md)
```
# H-XXXX — <제목>
- From: <Claude|Codex>   To: 사람(형)
- Status: open
- 배경:   <왜 형의 판단이 필요한가>
- 선택지: <A / B / ...>  (+추천 있으면)
- 영향:   <결정이 미치는 범위 (금전·보안·운영)>
```
